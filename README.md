# netSpy


<br>

## Requirement 
```
pip install scapy
pip install pandas
```
<br>

## Instructions


1. install requirements
2. Enable monitor mode in your network interface

```
$ airmon-ng start [interface name]
```

3. Verify your interface name

```
$ iwconfig
```

4. run ./netSpy.py -h 


<br>

## Usage

```
$ ./netSpy.py -s 
```
